# **🎵 Smart Music Downloader** #

Hey there! 👋 If you love discovering new songs, this little app is for you. Smart Music Downloader makes it super easy to download YouTube songs as high-quality 320 kbps MP3s — no ads, no stress, just music.

#Python #MusicDownloader #Automation #CustomTkinter #FFmpeg #HighQualityAudio
## Features ##
- Paste a YouTube link and download your favorite song instantly 🎶
- Automatically fetches song title, artist, and thumbnail
- Shows live download progress and sends a notification when done ✅
- Converts audio to MP3 using FFmpeg
- Portable Windows .exe version available
  
https://github.com/user-attachments/assets/0db693ed-f95e-49f8-8eb3-814a112fe92c

## Dependencies ##
- Python packages: yt_dlp, CustomTkinter, Pillow
- FFmpeg for audio conversion (must be installed or included in the folder)

> Don’t worry — if you download the .exe, you don’t need Python or FFmpeg separately!

## Download the App ##
- Check out the Releases tab for the Windows .exe file
- Simply download and double-click to run

**Paste a YouTube link, click Download, and enjoy your music! 🎧**
